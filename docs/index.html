Usage Info

The input type is detected automatically on entry.

Input Types

  • Decimal: Positive integers up to 2^64 - 1 (u64::MAX)
  • Hexadecimal: Must start with 0x (e.g., 0xFF)
  • Text: ASCII printable characters
  • dollcode: ▖, ▘, and ▌ characters

Buffer Limits

  • Maximum input length: 60 bytes // ASCII characters
  • Text encoding: Each input character is encoded into 5 dollcode characters (3 byte Unicode characters), resulting in an output of 15 bytes
  • The output buffer is set to not be exceeded by inputs

What?

dollcode is a trinary number encoding system.

It is denoted by three bit states, which are:

▖ = 1 | ▘ = 2 | ▌ = 3

The most significant bit is on the left side.
